好的，我们来分别看看这些内容，以及它们在考研数学中可能涉及的拓展：

  1. 二次函数极值

基本形式： f(x) = ax² + bx + c (a ≠ 0) 极值点： x = -b / 2a 极值： 如果 a > 0，函数有最小值，最小值为 f(-b / 2a) 如果 a < 0，函数有最大值，最大值为 f(-b / 2a) 拓展内容（考研常见）： 含参数的二次函数： 讨论参数变化对极值点和极值的影响。例如，给定一个含参数的二次函数，求其在某个区间上的最大值或最小值，这需要考虑极值点是否在区间内，以及区间端点值。 二次函数与不等式： 利用二次函数的图像和性质解决不等式问题。例如，ax² + bx + c > 0 恒成立，则需要 a > 0 且判别式 Δ < 0。 二次函数与方程根： 二次方程的根与二次函数图像和 x 轴的交点密切相关。判别式 Δ = b² - 4ac 决定了根的个数。 二次函数模型的应用： 将实际问题转化为二次函数模型，求解最大值或最小值。 2. 三次函数单调性

基本形式： f(x) = ax³ + bx² + cx + d (a ≠ 0) 导数： f'(x) = 3ax² + 2bx + c 单调性判断： f'(x) > 0，函数单调递增 f'(x) < 0，函数单调递减 f'(x) = 0，可能为极值点（需要进一步判断） 拓展内容（考研常见）： 单调区间： 求解 f'(x) > 0 和 f'(x) < 0 的不等式，确定单调区间。 极值点： f'(x) = 0 的根是可能的极值点。需要通过二阶导数或一阶导数符号的变化来判断是极大值还是极小值。 含参数的三次函数： 讨论参数变化对单调性和极值的影响。这通常需要分析导函数的根的情况。 三次函数与方程根： 三次方程的根与三次函数图像和 x 轴的交点相关。可以通过分析函数的单调性和极值来判断根的个数。 不等式证明： 利用三次函数的单调性证明不等式。例如，要证 f(x) > g(x)，可以构造函数 h(x) = f(x) - g(x)，然后证明 h(x) > 0。 3. 一次函数单调性

基本形式： f(x) = kx + b 单调性判断： k > 0，函数单调递增 k < 0，函数单调递减 k = 0，函数为常数 拓展内容（考研相对简单）： 分段一次函数： 讨论分段点处的连续性和单调性。 绝对值函数： f(x) = |kx + b| 可以看作是分段一次函数，需要分情况讨论。 线性规划： 一次函数是线性规划的基础。 考研数学中的注意事项：

导数是核心： 单调性和极值问题通常需要求导。 数形结合： 画图可以帮助理解函数的性质。 参数讨论： 含参数的问题是考研的重点。 综合应用： 这些知识点经常与其他知识点结合起来考察，例如不等式、方程、数列等。 希望这些内容对你有所帮助！ 如果你有具体的问题，可以随时提出。
